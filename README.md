### Assignment 1 : SVM and ANN with Raw Pixel, HoG, and LBP for handwritten symbol classification

1.	Assignment_1_ANN.ipynb = consists of ANN models for using with Raw Pixels, HoG features, and LBP features
2.	Assignment_1_Feat_Ext.ipynb = consists of Feature Extraction process for HoG and LBP features along with splitting of dataset into training and testing sets for Raw Pixels, HoG and LBP features. These extracted features are stored in the form of .npy files in the features/ directory and the split sets are stored in train/ and test/ directories.
3.	Assignment_1_Preprocessing.ipynb = consists of initial steps for extracting, loading, and exploring the original dataset. The images and labels are stored in .npy files and added to original_data/ directory
4.	Assignment_1_SVM.ipynb = consists of ANN models for using with Raw Pixels, HoG features, and LBP features
5.	Metadata.json = Metadata from original dataset
6.	Student_25675238 = Original extracted dataset
7.	Student_25675238.zip = Original compressed dataset
